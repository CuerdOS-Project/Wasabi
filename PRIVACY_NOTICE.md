# Políticas de Privacidad y Compromiso Ético: El Ecosistema Este software

Este documento establece las directrices fundamentales bajo las cuales opera el desarrollo de **Este software**, la aplicación de bienvenida de última generación. Como parte del ecosistema de software libre, nuestra misión es garantizar la transparencia, la libertad del usuario y la protección de sus datos.

---

## 1. Política de Privacidad en Software Libre
En Este software, entendemos la privacidad no como un privilegio, sino como un derecho fundamental. A diferencia del software privativo, nuestra política se basa en principios de "privacidad por diseño" y "privacidad por defecto":

* **Minimización de Datos:** Este software no recopila información personal identificable. Cualquier dato recolectado para fines de telemetría (si es activada explícitamente por el usuario) es anonimizado y no puede ser vinculado a una identidad específica.
* **Transparencia Total:** Dado que el código fuente es auditable, cualquier usuario puede verificar por sí mismo qué datos se envían, a dónde y con qué propósito. No existen "puertas traseras" ocultas.
* **Sin Rastreo:** Nos comprometemos a no integrar bibliotecas de terceros que realicen seguimiento de comportamiento o publicidad dirigida.

## 2. Compromiso de Código Siempre Libre y Neutral
Este software se adhiere estrictamente a los pilares de la Free Software Foundation (FSF). Nuestro compromiso es:

* **Libertad Total:** El usuario es dueño de su experiencia. Se garantiza el derecho a ejecutar, estudiar, compartir y modificar el programa según las necesidades del usuario.
* **Neutralidad Tecnológica:** El código es agnóstico respecto a intereses corporativos. No favorecemos a ninguna entidad, formato propietario o plataforma específica. El desarrollo está impulsado por la comunidad y la excelencia técnica, no por modelos de negocio que comprometan la integridad del software.
* **Ética de Colaboración:** Rechazamos cualquier forma de "tivo-ización" (hardware que impide el uso de software modificado). Este software permanecerá abierto para cualquier arquitectura o entorno.

## 3. Notas de Legalidad bajo la Licencia Pública General (GPL)
Este software es un proyecto licenciado bajo la **GNU General Public License (GPL)**, preferiblemente en su versión 3 o superior. Esto implica compromisos legales irrevocables para los colaboradores y usuarios:

1.  **Copyleft:** Cualquier obra derivada de Este software debe ser distribuida bajo los mismos términos de la licencia GPL. Esto asegura que la libertad del código se mantenga de forma perpetua.
2.  **Disponibilidad del Código Fuente:** Al distribuir copias del software (modificadas o no), es una obligación legal proveer acceso al código fuente completo correspondiente.
3.  **Renuncia de Garantía:** Tal como se estipula en la cláusula 15 y 16 de la licencia GPL, el software se entrega "tal cual", sin garantías de ningún tipo por parte de los desarrolladores originales, limitando nuestra responsabilidad ante errores o daños derivados del uso del software.
4.  **Propiedad Intelectual:** Los colaboradores ceden sus contribuciones al proyecto manteniendo su autoría, bajo el entendido de que dichas contribuciones se integran en un cuerpo de trabajo bajo licencia GPL, permitiendo la explotación libre y abierta del mismo.
